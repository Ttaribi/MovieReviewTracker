# Movie Review Tracker

A web application that lets users search for movies using the **OMDb API** and view their own reviews, which are stored and retrieved from a **MongoDB** database.

## Live Demo
- **Deployed App:** [https://moviereviewtracker.onrender.com ](https://www.omdbapi.com/) 
- **YouTube Demo:** [https://youtu.be/nt5IF0x4APg](https://youtu.be/nt5lF0x4APg)  

## API Used
- **Open Movie Database (OMDb) API:** https://www.omdbapi.com/

## Key Features
- Search for movies by title using the OMDb API
- View movie results/details returned by the API
- Create reviews for movies and store them in MongoDB
- View saved reviews from the database
- **Delete reviews** when you no longer want them

## Tech Overview
- External movie data: **OMDb API**
- Review storage: **MongoDB**
- Web app (frontend + backend) that connects search results with stored user reviews

## Getting Started (Local Setup)

### Prerequisites
- **Node.js** (recommended: latest LTS)
- **MongoDB** (local MongoDB or MongoDB Atlas)
- **OMDb API key** (get one from OMDb)
